# Role-Based RAG Chatbot

This project implements a Retrieval-Augmented Generation (RAG) chatbot with role-based access control (RBAC).

## Current Features
- Multi-folder document ingestion
- Markdown and CSV support
- Text chunking and embedding
- Vector database indexing using Chroma
- Role-tagged metadata for secure retrieval
- Semantic validation scripts

## Next Steps
- FastAPI chat API
- YAML-based RBAC policy engine
- LLM integration
- React frontend
